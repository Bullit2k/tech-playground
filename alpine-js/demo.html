Alpine.js Cheat Sheet

Explore key Alpine.js features with explanations and the corresponding code snippets.

1. Basic Data Binding (`x-data` and `x-text`)

Alpine.js lets you bind data to the DOM using `x-data` for reactive data and `x-text` to display it.

Output:

Show/Hide Code

<div x-data="{ message: 'Hello Alpine!' }">
  <p x-text="message"></p>
  <input x-model="message" placeholder="Type something">
</div>

2. Conditional Rendering (`x-show`)

Toggle the visibility of elements using `x-show`. It keeps the element in the DOM but hides it.

This content is toggled using x-show.

Show/Hide Code

<div x-data="{ showContent: true }">
  <button x-on:click="showContent = !showContent">Toggle Content</button>
  <div x-show="showContent">
    <p>This content is toggled using <strong>x-show</strong>.</p>
  </div>
</div>

3. Conditional Rendering (`x-if`)

Use `x-if` to conditionally render or remove elements from the DOM completely.

Show/Hide Code

<div x-data="{ showIf: true }">
  <button x-on:click="showIf = !showIf">Toggle Content</button>
  <template x-if="showIf">
    <div>
      <p>This content is rendered conditionally using <strong>x-if</strong>.</p>
    </div>
  </template>
</div>

4. Looping with `x-for`

Loop through an array of items with `x-for` and dynamically render elements.

Show/Hide Code

<div x-data="{ items: ['Item 1', 'Item 2', 'Item 3'] }">
  <ul>
    <template x-for="(item, index) in items" :key="index">
      <li x-text="item"></li>
    </template>
  </ul>
</div>

5. Event Binding (`x-on`)

Listen for DOM events like clicks using `x-on` and respond by updating data or calling functions.

Click count:

Show/Hide Code

<div x-data="{ count: 0 }">
  <button x-on:click="count++">Click Me</button>
  <p>Click count: <strong x-text="count"></strong></p>
</div>

6. Two-way Data Binding (`x-model`)

Use `x-model` to create two-way data binding between input fields and your Alpine.js data.

Hello, !

Show/Hide Code

<div x-data="{ name: '' }">
  <input x-model="name" placeholder="Enter your name">
  <p>Hello, <strong x-text="name"></strong>!</p>
</div>
